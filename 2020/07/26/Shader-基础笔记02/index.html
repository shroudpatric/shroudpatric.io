顶点/片元的调用方式

  • 需要注意的是，定点着色器是逐顶点调用的，片元着色器是逐片元调用的。片元着色器中的输入实际上是把定点着色器的输出进行插值后得到的结果。

ShaderLab 和 CG 的变量类型对应

ShaderLab 属性类型 CG 变量类型
Color，Vector float, half4, fixed4
Range, Float float, half, fixed
2D sampler2D
Cube samplerCube
3D sampler3D
